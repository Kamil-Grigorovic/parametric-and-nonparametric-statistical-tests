# Parametric and Non-Parametric Statistical Tests

This project presents a statistical analysis of student productivity data obtained from Kaggle. The analysis includes data exploration, visualization, descriptive statistics, and hypothesis testing using both parametric and non-parametric methods.

## Dataset
Ultimate Student Productivity Dataset (Kaggle)

## Topics Covered
- Descriptive statistics
- Data visualization
- Normality testing
- Parametric tests
- Non-parametric tests
- Hypothesis testing
- Statistical inference

## Technologies
- R
- R Markdown
- ggplot2
- dplyr
- tidyr
